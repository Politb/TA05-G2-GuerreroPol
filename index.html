Fail2ban + Discord

This project integrates Fail2Ban with Discord to provide real‑time security alerts. The system detects brute‑force and DoS attacks on services like SSH and Apache, automatically blocks malicious IPs, and sends instant notifications to a Discord channel via Webhooks.
